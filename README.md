# DTC Student System

A web-based Student Management System developed for school requirements.

## 📌 Features
- User login system (Admin, Registrar, Student)
- Student dashboard
- Add / View / Delete records
- Role-based access control
- Database integration using MySQL

## 🛠️ Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP / Localhost environment

## 📂 Project Structure
- /admin - Admin panel
- /student - Student dashboard
- /registrar - Registrar module
- /assets - CSS, JS, images
- /database - SQL file
